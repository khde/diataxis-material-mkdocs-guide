# Diataxis Material for MkDocs Guide

Documentation project using [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) and structured with the [Diátaxis](https://diataxis.fr/) framework.

## Quick Start

1. Create a virtual environment:

	```bash
	python3 -m venv .venv
	```

2. Activate the virtual environment:

	```bash
	source .venv/bin/activate
	```

3. Install dependencies:

	```bash
	pip install -r requirements.txt
	```

4. Start the docs server:

	```bash
	mkdocs serve
	```

Then open the local URL shown in the terminal.
